# 🎵 Audio Converter Pro

## ✨ Features

- 🎯 **Multiple Formats**: MP3, FLAC, WAV, OGG, M4A, Opus, and more
- 🚀 **Fast Conversion**: Multi-threaded processing
- 📋 **Batch Processing**: Convert entire folders at once
- 🎨 **Modern UI**: Dark/Light themes with instant switching
- 📊 **File Info**: Detailed audio information panel
- 📜 **History**: Track all your conversions
- 🔧 **Presets**: Quick settings for common scenarios
- 🖱️ **Drag & Drop**: Just drop files to convert
